Welcome to Yuanchao AWS Hands-on project

The project's learning path sorted by incresed complexity and amount of used resources: 1->2->3->6->4->5->8->7->9

The goal of this project aims to: 1.lean all most commonly used AWS resources 2.use these AWS resoucres the same way taht they are implemented in industry's real production/work environment 3.have an automated process that can reprodure each project just by almost one click.

As they are involved in almost every project, AWS IAM and AWS Cloudwatch are not mentioned in this intruduction page.

  • Topic 1 Simple Http Static Website(S3).
  • Topic 2 Https Static Website with a public DNS domain with(S3+Route 53+Certificate Manager+CloudFront).
  • Topic 3 Shared FS between 2 servers(EC2+EFS).
  • Topic 4 Docker image build's CI/CD, push to private DTR(Codebuild+ECR).
  • Topic 5 Docker container website(service) using Database(ECS+FARGATE+RDS).
  • Topic 6 Simple http website hosted on loadbalancing & autoscaling private backend(ELB+EC2+VPC+Auto-scaling).
  • Topic 7 RESTful application(API) using Database(DynamoDB+SES+ApiGateway+StepFunctions+Eventbridge+Lambda).
  • Topic 8 Scheduler batch job application(Batch).
  • Topic 9 in progress.
